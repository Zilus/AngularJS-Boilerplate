# AngularJS
Boilerplate for AngularJS projects

## 1. Setup
```bash
npm install
```

## 2. Watch files
```bash
gulp
```

## 3. Build production version
```bash
gulp build
```

## 4. Start webserver without watch task
```bash
gulp server
```

## 5. Start webserver from build folder
```bash
gulp server-build
```
